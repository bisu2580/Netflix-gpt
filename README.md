# Netflix GPT

- This is an app made as a clone of Netflix and some added features.
- The app is built with React and Vite.
- Header
- Routing of App
- Login Page
- Sign Up Form
- Form Validation
- useRef Hook
- Firebase Setup
- Deploying App to production

# Features

- Login and Signup Page
  - Sign In Sign Up Form
  - Redirect
- Browse (after authentication)
  - Header
  - Main Movie
    - Trailer in Bg
    - Title and desc
    - Movie suggesions
      - MoviesList
- NetflixGPT
  - Search Bar
  - Movie Suggestions
